# Melody v. BCSD

A citizen-journalism archive for *Georgia Trust for Local News LLC d/b/a The Macon Melody v. Bibb County School District*, Civil Action No. 2025-CV-083495, Bibb County Superior Court (State of Georgia).

**Live site:** https://civicpulse.github.io/Melody-v-BCSD/

This repository holds:

- The 25 court filings (`/pdfs/`) — sourced from Tyler re:SearchGA.
- A page per filing (`/filings/`) — plain-English overview, deep dive for non-lawyers, and a link to the original PDF.
- A long-form Case Report, Timeline, Parties & Counsel, Legal Background, and Glossary (`/docs/`).
- A Jekyll site built on the [just-the-docs](https://just-the-docs.com/) theme for GitHub Pages.

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000/Melody-v-BCSD/.
