# References

Every legal authority, statute, regulation, and secondary source cited on this site, with links to the full text and a list of pages where each reference appears.

{: .note }
> **How to read legal citations:** Case citations follow the format *Case Name*, volume / reporter / page (year). Statute citations use the format Title § Section. "O.C.G.A." = Official Code of Georgia Annotated. "U.S.C." = United States Code. "C.F.R." = Code of Federal Regulations. "Ga." = Georgia Supreme Court reporter. "Ga. App." = Georgia Court of Appeals reporter. "F. Supp." = Federal Supplement (federal trial court decisions). "U.S." = United States Reports (U.S. Supreme Court).
